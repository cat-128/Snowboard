# Snowboard
It's a soundboard like the one you find in discord but you can use it anywhere. pretty cool.. <br>
Still in development

Features -
- You can add as many sound as you like
- completely opensource
- custom keybinds
- can be used in most* applications

> note: the code may not be that clean.

To compile you need the following libraries ->
- SDL 3
- SDL3 TTF
- SDL3 mixer
- SDL3 image
